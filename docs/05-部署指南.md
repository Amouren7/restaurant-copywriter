# 餐饮文案助手 — 部署指南

> 从零到上线，预计30分钟。

---

## 前置准备

你需要准备以下账号（全部免费注册）：

| 账号 | 用途 | 注册地址 |
|------|------|---------|
| GitHub | 存放代码 | https://github.com |
| Railway | 部署网站 | https://railway.app |
| DeepSeek | AI接口 | 你已有API Key ✅ |

---

## 第一步：上传代码到GitHub（10分钟）

### 1.1 创建仓库

1. 登录 GitHub，点击右上角 "+" → "New repository"
2. Repository name 填：`restaurant-copywriter`
3. 选 **Public**（免费用户必须Public才能被Railway部署）
4. 点击 "Create repository"

### 1.2 上传代码

**方式一：网页上传（最简单，不需要装Git）**

1. 进入刚创建的仓库页面
2. 点击 "uploading an existing file"
3. 把以下文件/文件夹拖进去：
   - `src/` 整个文件夹
   - `Procfile`
   - `runtime.txt`
4. 点击 "Commit changes"

**方式二：命令行上传（如果你会用Git）**

```bash
cd "D:\CC project\restaurant-copywriter"
git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/你的用户名/restaurant-copywriter.git
git push -u origin main
```

---

## 第二步：部署到Railway（10分钟）

### 2.1 注册Railway

1. 打开 https://railway.app
2. 点击 "Login" → 用GitHub账号登录（推荐）
3. 授权Railway访问你的GitHub

### 2.2 创建项目

1. 点击 "New Project"
2. 选择 "Deploy from GitHub Repo"
3. 选择 `restaurant-copywriter` 仓库
4. Railway会自动开始部署

### 2.3 配置环境变量

部署开始后，需要配置API密钥：

1. 点击你的项目卡片
2. 点击 "Variables" 标签
3. 点击 "New Variable"，添加以下变量：

| Variable Name | Value |
|---------------|-------|
| `DEEPSEEK_API_KEY` | 你的DeepSeek API密钥 |
| `DEEPSEEK_BASE_URL` | `https://api.deepseek.com` |

4. 添加完后，Railway会自动重新部署

### 2.4 获取域名

1. 点击 "Settings" 标签
2. 找到 "Networking" 部分
3. 点击 "Generate Domain"
4. Railway会给你一个域名，类似：`restaurant-copywriter-production-xxxx.up.railway.app`

**这就是你的网站地址！**

## 第三步：测试上线（5分钟）

1. 用手机浏览器打开你的域名
2. 测试完整流程：
   - [ ] 首屏正常显示，效果对比清晰

   - [ ] 点复制按钮，文案成功复制
   - [ ] 用完5次后，显示次数用完提示
3. 如果一切正常 🎉 网站已上线！

## 常见问题

### Q：部署失败怎么办？

查看Railway的 "Deployments" 标签，点击最新的部署记录查看日志。
常见原因：
- 环境变量没填对
- `requirements.txt` 格式问题
- Python版本不兼容

### Q：网站打开了但生成文案报错？

检查：
1. DeepSeek API Key是否正确
2. DeepSeek账户余额是否充足
3. 查看Railway日志中的具体错误信息

### Q：Railway免费额度够用吗？

Railway免费计划：
- 每月 $5 的免费额度
- 按使用量计费（CPU+内存+流量）
- 一个小型Flask应用每月约 $1-2
- **完全够用，不需要花钱**

### Q：怎么更新网站？

修改代码后，push到GitHub，Railway会自动重新部署。
或者在Railway控制台点击 "Redeploy"。
