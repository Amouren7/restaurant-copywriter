## 四、技术要点

### 4.1 启动方式

本地开发：
```bash
cd "D:\CC project\restaurant-copywriter\src"
export DEEPSEEK_API_KEY="sk-你的key"
export DEEPSEEK_BASE_URL="https://api.deepseek.com"
export DEEPSEEK_MODEL="deepseek-v4-flash"
python server.py
```

Windows也可双击 src/run_local.bat（需先修改API Key）。

生产部署：详见 docs/05-部署指南.md

### 4.2 环境变量

| 变量名 | 必填 | 说明 |
|--------|------|------|
| DEEPSEEK_API_KEY | 是 | API密钥 |
| DEEPSEEK_BASE_URL | 是 | https://api.deepseek.com |
| DEEPSEEK_MODEL | 否 | 默认 deepseek-v4-flash |

### 4.3 API接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | / | 首页 |
| GET | /api/categories | 品类列表 |
| GET | /api/scenes | 场景列表 |
| POST | /api/generate | 生成文案（核心） |
| POST | /api/verify | 验证激活码（预留） |
| GET | /api/health | 健康检查 |

### 4.4 提示词系统（src/prompts.py）

核心函数 build_prompt() 根据品类+场景+参数构造完整提示词。
设计原则：口语化、接地气、带emoji、禁止编造虚假信息。

### 4.5 免费限制

浏览器 localStorage 实现，每天5次，跨天重置。
代码位置：index.html 中 checkUsageLimit() / incrementUsage()
